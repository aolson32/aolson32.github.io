My Machine Learning Data Science project looked at the usage of Spatial Dropout layers instead of standard dropout layers; it tested to see if this would increase testing accuracy.

Report Writeup

As confirmed through the tests of this project, the use of Spatial Dropout 2D layers did not increase the testing results scores over the use of standard Dropout layers.
